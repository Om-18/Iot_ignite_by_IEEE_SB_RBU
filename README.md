# ⚡ IoT Ignite – IEEE SB RBU

This repository contains the codes used in the **IoT Ignite Workshop** conducted by the **IEEE Student Branch, Ramdeobaba University (RBU)**.

The workshop introduces participants to basic **IoT sensors and actuators** using microcontrollers.

---

## 🔧 Modules Covered

### 1️⃣ Touch Sensor

* Learn how capacitive touch sensors work
* Detect touch input and trigger actions

### 2️⃣ Gas Sensor

* Detect gas or smoke in the environment
* Useful for safety and monitoring applications

### 3️⃣ IR Sensor

* Detect obstacles or nearby objects
* Commonly used in automation and robotics

### 4️⃣ Servo Motor

* Control angular motion of a motor
* Used in robotics, automation, and IoT devices
